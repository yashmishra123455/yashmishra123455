## 🏆 Featured Projects

### 🚨 [Women Safety System](https://github.com/yashmishra123455/women-safety)
An AI-powered safety solution using **hand gesture recognition** that instantly sends help requests to the nearest police station and alerts emergency contacts within seconds. Built with computer vision and real-time notification systems to ensure rapid response in critical situations.

**Tech Stack:** Python, OpenCV, Machine Learning, Geolocation API, SMS Gateway

### 🥗 [NutriScan - Smart Food Analyzer](https://github.com/yashmishra123455/nutriscan)
An intelligent nutrition analyzer that scans food items and provides comprehensive health insights. Features **barcode scanning**, ingredient analysis, personalized health alerts for conditions like high blood pressure, thyroid, and diabetes, plus AI-powered healthy food recommendations.

**Tech Stack:** Python, Computer Vision, ML, Barcode Recognition API, Nutrition Database

### 🌾 [AgriSmart - Farmer's AI Assistant](https://github.com/yashmishra123455/agrismart)
A comprehensive agricultural platform providing farmers with **plant disease detection** for crops (tomato, potato, beans, corn, rice), **weather forecasting**, and disaster alerts. Sends real-time SMS notifications about severe weather conditions like tornadoes and heavy rainfall to help farmers protect their crops.

**Tech Stack:** Python, Machine Learning, Weather API, SMS Gateway, Image Classification

### ♟️ [AI Chess Engine](https://github.com/yashmishra123455/chess-ai)
A fully-functional chess game supporting **multiplayer (1v1)** and **AI opponent** with adjustable difficulty levels. Features intelligent move prediction, game state management, and strategic AI algorithms for challenging gameplay.

**Tech Stack:** Python, Minimax Algorithm, Alpha-Beta Pruning, Game Development
